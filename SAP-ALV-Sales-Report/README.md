# SAP ALV Sales Report (BTP ABAP)

## 📌 Project Overview
This project demonstrates a simple Interactive Sales Report using ALV in SAP BTP ABAP Environment.

## 🚀 Features
- Display sales data in ALV format
- Filter by customer
- Calculate total sales
- Clean and simple UI

## 🛠 Tech Stack
- SAP BTP ABAP Environment
- ABAP (Advanced Business Application Programming)
- ALV (ABAP List Viewer)
